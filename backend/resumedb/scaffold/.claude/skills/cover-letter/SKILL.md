description: Draft a cover letter or short-answer responses for the current application. Use when the user asks for a cover letter or answers to application questions.
---

Draft cover-letter.md (or short-answer text) in the current application folder.

- Source material: db/ entries, db/profile.yaml, jd.md, notes.md, and the
  tailored resume.yaml. Everything must trace to a db/ fact - the truth policy
  from tailor-resume applies verbatim.
- Voice: plain, direct, specific. No "I am writing to express my interest",
  no adjectives doing the work facts should do. Lead with the strongest
  role-relevant fact.
- Length: under one page, 3-4 short paragraphs, unless the user asks otherwise.
- Company specifics come only from jd.md and notes.md; do not guess company
  details that are not in the folder.
- Short-answer questions: answer in the same voice, respecting any stated word
  limits.
